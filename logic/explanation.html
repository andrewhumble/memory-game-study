Task Performance, Emotions, and Heart Rate as Indicators for Perceived Stress

In this experiment, you will be asked to solve word puzzles with different levels of difficulty. Before that you will be placed in one of three groups: no support through adaptation, task performance, or AI-based analysis of the camera image.

What can be Inferred from the Task Performance?

The task performance provides details about your individual processing efficiency. If you solve tasks in a sequence correctly or incorrectly, this is an indicator of good or bad performance. The task difficulty will be adjusted accordingly to match the current performance level. For example, tasks will become easier if many tasks were previously answered incorrectly and tasks will become harder if many tasks were previously answered correctly.

Which Details are Analyzed with a Camera Image?

An artificial intelligence (AI) captures the face and provides details about stress levels and emotional states. The stress level is calculated by analyzing the face to detect changes in blood circulation and pulse [1]. Recent scientific research has shown that pulse changes are related to the current level of perceived stress [2]. Emotional states can also be derived from facial features (smiling, frowning), which are indicative of the current affective state [3]. Using these two indicators, task difficulty can be inferred and adjusted to avoid frustration due to tasks that were too difficult or too easy using an AI.
